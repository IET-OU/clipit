********************************
** SQLSpaces Logging for elgg **
********************************

Installation:
Simply copy this folder to the "mod" folder of your elgg installation.
Upon deployment you should see a section on the Administration View called SQLSpaces. 
On this page you need to enter the socket for the server and the name of the space you want to log tuples to.
Afterwards you can enable or disable the plugin by submitting the form.

Have fun!
