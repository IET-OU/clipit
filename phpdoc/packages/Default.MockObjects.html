      SimpleCallSchedule¶

      Allows setting of actions against call signatures either at a specific time, or always. Specific time settings trump lasting ones, otherwise the most recently added will mask an earlier match.

      « More »

      SimpleMock¶

      A base class or delegate that extends an empty collection of methods that can have their return values set and expectations made of the calls upon them. The mock will assert the expectations against it's attached test case in addition to the server stub behaviour or returning preprogrammed responses.
